# VoltSwap - Silver-Ion Battery Swapping Platform Interface

## Overview

VoltSwap is a modern web-based interface for a silver-ion battery swapping platform designed for electric vehicles (EVs). This interface provides users with an intuitive dashboard to manage battery subscriptions, locate swapping stations, and track usage.

## Features

### Dashboard
- **Subscription Overview**: Displays current plan details, remaining swaps, and usage progress.
- **Quick Actions**: Easy access to common functions like finding stations, viewing history, and managing subscriptions.
- **Nearby Stations**: List of available battery swapping stations with availability status.
- **Recent Swaps**: History of recent battery swapping activities.

### Map View
- Interactive map showing station locations with color-coded availability (green for available, amber for moderate, coral for low).
- Sidebar with detailed station information including battery counts and distances.
- Search functionality to find specific stations.

### Subscription Management
- View and compare different subscription plans.
- Plans include varying numbers of swaps per month with different pricing tiers.
- Popular plan highlighting and selection interface.

### Authentication & Onboarding
- Secure login and registration forms.
- Guided onboarding process for new users.

## Technical Details

- **Frontend**: Pure HTML, CSS, and JavaScript
- **Styling**: Custom CSS with modern design system
- **Responsive**: Mobile-friendly interface with adaptive layouts
- **Animations**: Smooth transitions and micro-interactions

## Usage

1. Open `voltswap.html` in a modern web browser
2. Navigate through the interface using the sidebar menu
3. Use the dashboard for quick access to features
4. Explore the map to find nearby stations
5. Manage subscriptions through the dedicated section
